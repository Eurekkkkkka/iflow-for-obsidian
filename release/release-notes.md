## v0.8.0 更新内容

### 🏗️ 架构重构（重大更新）

本版本进行了完整的架构重构，从扁平结构升级为分层架构 (DDD)，大幅提升代码质量和可维护性。

**架构变化**：
- **分层架构**：应用层 (`app/`)、领域层 (`domain/`)、功能层 (`features/`)、基础设施层 (`infra/`)、UI层 (`ui/`)
- **TypeScript 文件**：从 7 个增加到 35 个
- **测试覆盖**：新增 21 个单元测试文件，覆盖率约 60%
- **测试框架**：引入 Vitest

**新增目录结构**：
```
src/
├── app/                    # 应用层 - 插件引导、SDK 管理
├── domain/                 # 领域层 - 业务逻辑、配额策略
├── features/               # 功能层 - 用例编排、流程控制
├── infra/                  # 基础设施层 - ACP 协议、平台适配
├── ui/                     # UI层 - 视图组件、交互逻辑
└── i18n/                   # 国际化
```

### ✨ 新增功能

| 功能 | 说明 |
|------|------|
| 存储配额警告 | 监控 localStorage 使用量，超限警告 |
| 连接诊断 | 运行时连接状态检测与报告 |
| 图片附件 UI | 支持图片附件显示与交互 |
| Canvas 文件解析 | 支持 Obsidian Canvas 文件 |
| 工具调用可视化 | 展示工具调用过程与结果 |
| 滚动位置持久化 | 保持对话滚动位置 |

### 🔧 技术改进

- **SDK 启动方式优化**：直接调用 Node.js 入口，不再弹出 Windows 控制台窗口
- **Bootstrap 模式支持**：更灵活的插件初始化
- **依赖注入模式**：更好的解耦和可测试性
- **平台适配抽象层**：更好的跨平台兼容性
