## v0.7.7 更新内容

### 🐛 关键 Bug 修复
- **修复流式响应文字重复问题**：修复了 AI 回复中每个字符被重复输出 2-3 次的问题
  - 根本原因：`on/off` 处理器注册机制有 bug，包装函数无法被正确移除
  - 解决方案：重构处理器管理，使用 `Map` 追踪处理器，新增 `clearHandlers()` 方法

### ✨ 新功能
- **自动启动 SDK**：插件加载时自动检测并启动 iFlow SDK
  - 设置中新增"自动启动 SDK"开关（默认开启）
  - 设置中新增"重启 SDK"按钮
  - 命令面板新增"重启 iFlow SDK"命令
- **思考中动画**：等待 AI 回复时显示跳动的小圆点 + "思考中..." 文字

### 🎨 UI 增强
- **Markdown 渲染增强**：
  - 代码块：带边框、语法高亮支持
  - 行内代码：紫色高亮背景
  - Thinking 块：浅紫色背景 + 左边框 + 斜体样式
  - 标题：H2/H3/H4 不同大小
  - 图标样式：✅❌🔄◆ 带颜色
  - 文件路径：代码风格高亮
  - 快捷键：键盘按键样式

### 🔧 技术改进
- 处理器管理重构，防止消息重复
- 样式系统优化，支持更多 Markdown 元素
- 新增 SDK 进程管理（spawn detached process）
