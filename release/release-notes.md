## v0.8.1 更新内容

### 🚀 新功能

- **终端命令支持**：新增 ACP 协议 `terminal/*` 方法实现，支持 Shell 命令执行
  - `terminal/create` - 创建终端会话并执行命令
  - `terminal/output` - 获取终端输出
  - `terminal/wait_for_exit` - 等待命令完成
  - `terminal/kill` - 终止运行中的命令
  - `terminal/release` - 释放终端资源

### 🐛 Bug 修复

- **Skill 执行失败问题**：修复在 Obsidian 插件中使用需要 Shell 命令的 Skill（如 `voice-transcriber`）时无法执行的问题
- 现在插件可以正确处理需要运行 Python 脚本或其他系统命令的 Skill

### 🔧 技术改进

- 更新 `clientCapabilities` 声明，添加 `terminal` 能力
- 使用 `child_process.spawn` 实现跨平台命令执行（支持 Windows PowerShell 和 Unix Bash）
- 添加终端会话管理，支持多个并行命令执行

## v0.8.0 更新内容

### ✨ 体验改进

- **运行状态可见**：顶部栏实时展示 SDK 连接状态（绿/黄/红），不再猜测服务是否在线
- **上下文透明化**：发送前看到将被传入模型的文件列表，可逐个移除
- **思考流与回答分离**：AI 推理过程单独展示为折叠块，不与最终回答混流
- **工具调用可视化**：每个工具执行独立卡片，显示参数/状态/结果，完成后自动折叠

### 🎨 视觉精修

- 所有图标按钮补齐键盘焦点环（Tab 键可访问）
- 7 个操作按钮补充中文无障碍标签（屏幕阅读器友好）
- 动画改为具体属性过渡，消除 hover 时的额外抖动

### 🛡️ 诊断与恢复

- 设置页"诊断与状态"区展示 SDK 端口、连接状态和存储占用
- 新增"重新检测"按钮独立于"重启 SDK"，支持快速验连
- 存储接近配额（>80%）时主动弹出 Notice 提示

### 🔧 行为说明

- 思考模式开启时，AI 推理不再与回答混排，两者独立区域展示
- 工具调用完成后自动折叠，不遮挡主对话内容
- 存储配额警告在同一会话内仅展示一次，不重复打扰

## v0.7.8 更新内容

### 🎨 UI/UX 改进
- **发送按钮增强**：增大按钮尺寸，添加渐变色和阴影效果，悬停时微微上浮
- **输入框优化**：添加右侧内边距，防止文字被发送按钮遮挡
- **动态问候语**：根据时间和日期显示不同的欢迎消息
  - 早上好（5-9点）、上午好、中午好、下午好、晚上好、夜深了
  - 周五快乐、周六好、周日愉快等特殊问候
  - Emoji 添加摇摆动画，问候文字使用渐变色

### 🐛 Bug 修复
- **会话列表显示问题**：修复只显示"今天"的对话，现在按日期分组显示所有对话（今天、昨天、本周、更早）
- **编号不连续问题**：AI 生成内容时列表编号从 1 开始，避免出现 15、16、17 等不连续编号

### 🔧 技术改进
- 添加 i18n 翻译：yesterday、thisWeek、older
- 优化会话列表渲染逻辑，按更新时间排序
