# iFlow for Obsidian — 人工回归清单

> 适用版本：v0.8.0+  
> 使用方式：每次发布前按场景逐项执行，记录通过 ✅ / 失败 ❌ / 跳过 ⏭。

---

## 1. 首次启动

| # | 步骤 | 预期结果 |
|---|------|----------|
| 1.1 | 安装插件后首次打开聊天面板 | 显示动态问候语（早上好/下午好等），无报错 |
| 1.2 | iFlow SDK 未运行时打开面板 | 顶部状态栏显示"未连接"状态点 |
| 1.3 | 启动 iFlow SDK 后等待或点击"重新检测" | 状态变为"已连接"，绿色状态点 |
| 1.4 | 打开设置页 → 诊断与状态 → 查看报告 | 异步展示端口、连接状态和存储用量 |
