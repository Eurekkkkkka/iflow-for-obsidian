# 模块开发文档：messageRenderer

## 1. 模块定位

`messageRenderer` 负责将聊天消息文本渲染为 HTML，承接 Markdown 风格格式化规则。

## 2. 核心职责

1. 处理代码块和行内代码。
2. 处理标题、列表、强调样式。
3. 处理工具图标和文件路径高亮。
4. 提供统一 HTML 转义。

## 3. 对外接口

1. `render(content: string): string`

## 4. 依赖关系

1. 无 Obsidian 依赖。
2. 可依赖 i18n 以支持局部文案。

## 5. 联通说明

1. `messageListView` 渲染消息时调用本模块。
2. 历史消息加载和流式消息更新都使用同一渲染入口。

## 6. 开发约束

1. 保持纯函数属性。
2. 不直接操作 DOM 节点。

## 7. 验收标准

1. 当前格式化表现不回归。
2. HTML 转义行为不回归。
