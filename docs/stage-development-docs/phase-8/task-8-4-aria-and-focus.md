# Task 8.4：补图标按钮的 aria-label 与 focus 态

## 所属阶段

阶段八：视觉精修与可访问性

## 目标

提升图标按钮和关键交互控件的可访问性，保证键盘和屏幕阅读器场景基本可用。

## 实施步骤

1. 盘点当前所有图标按钮。
2. 为缺失项补 aria-label。
3. 检查并统一 focus-visible 样式。
4. 做键盘可达性走查。

## 验收标准

1. 关键图标按钮均有清晰语义标签。
2. 键盘聚焦时有明确可见反馈。
3. 不影响现有视觉层级。

## 风险与注意事项

1. aria-label 文案必须准确简洁。
2. focus 样式要清晰但不过于刺眼。
