# Task 6.1：统一输入区快捷键策略

## 所属阶段

阶段六：高频交互效率优化

## 目标

明确发送、换行、停止生成等输入区核心快捷行为，减少用户试错。

## 实施步骤

1. 列出当前所有已存在快捷行为。
2. 确定最终保留的快捷键规则。
3. 定义与 Obsidian 全局快捷键的冲突规避策略。
4. 为不同状态下的快捷行为补说明。

## 验收标准

1. 输入区快捷行为规则明确且一致。
2. 用户无需反复试错即可完成常见操作。
3. 不与 Obsidian 常见快捷键产生明显冲突。

## 风险与注意事项

1. 快捷键变更必须配套提示文案。
2. 不要引入过多快捷键增加记忆成本。
