# Task 3.1：修复代码块语言标签显示

## 所属阶段

阶段三：消息内容渲染与阅读质量

## 目标

让代码块语言标签稳定显示，帮助用户快速识别代码片段所属语言。

## 实施步骤

1. 盘点当前代码块渲染结构。
2. 为语言标识设计稳定的数据承载方式。
3. 调整样式展示方案。
4. 为有语言和无语言两种代码块补测试。

## 涉及文件

1. `src/ui/chat/messageRenderer.ts`
2. `src/styles.css`
3. `tests/messageRenderer.test.ts`

## 验收标准

1. 常见语言标签可正确显示。
2. 无语言代码块不会显示错误标签。
3. 深浅主题下标签可读性稳定。

## 风险与注意事项

1. 不要依赖脆弱的 CSS 类名解析。
2. 需要兼顾未来语法高亮扩展。
