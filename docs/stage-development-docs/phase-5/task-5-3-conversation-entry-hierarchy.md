# Task 5.3：重构当前会话、历史入口和新建入口的层级关系

## 所属阶段

阶段五：顶部信息架构重组

## 目标

让用户更清楚地区分“我当前在哪个会话”“怎么切历史”“怎么新建对话”。

## 实施步骤

1. 重新定义顶部主入口的优先级。
2. 决定当前会话标题的表达方式。
3. 明确历史面板和新建按钮的关系。
4. 验证首次使用者是否能快速理解。

## 验收标准

1. 顶部主操作路径更直观。
2. 历史入口与新建入口不再竞争注意力。
3. 当前会话状态更容易理解。

## 风险与注意事项

1. 会话入口重构会影响高频操作路径，必须谨慎。
2. 要兼顾无历史会话和有大量历史会话两种场景。
