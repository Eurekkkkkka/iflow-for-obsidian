# Task 2.5：为上下文摘要区增加操作能力

## 所属阶段

阶段二：运行时反馈与上下文可见性

## 目标

让用户可以直接在上下文摘要区撤销、移除或理解上下文来源，而不必通过隐式逻辑猜测。

## 核心动作

1. 移除当前上下文项。
2. 区分自动附加和手动附加来源。
3. 为不同来源提供说明文案。

## 实施步骤

1. 为上下文项定义操作按钮和点击目标。
2. 增加来源标签或说明。
3. 处理移除后的状态同步。
4. 补充必要的 hover 和 focus 态。

## 验收标准

1. 用户可以一键移除不想发送的上下文。
2. 移除后 UI 和内部状态保持一致。
3. 自动附加来源有清晰说明，不再显得像“偷偷发送”。

## 风险与注意事项

1. 不能因为操作增强而让摘要区过于拥挤。
2. 必须注意键盘可达性。
