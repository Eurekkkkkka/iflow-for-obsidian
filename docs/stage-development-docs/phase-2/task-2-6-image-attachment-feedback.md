# Task 2.6：增强图片附件反馈

## 所属阶段

阶段二：运行时反馈与上下文可见性

## 目标

让图片附件过程具备更强的可见性，包括数量、总大小和删除反馈。

## 实施步骤

1. 为附件区显示当前图片数量。
2. 计算并显示总大小。
3. 为单张图片删除提供明确反馈。
4. 为超限和不支持格式统一提示。

## 涉及文件

1. `src/chatView.ts`
2. `src/ui/chat/imageAttachmentView.ts`
3. `src/styles.css`

## 验收标准

1. 用户能知道当前已附加多少图片。
2. 用户能知道附件大小是否接近上限。
3. 删除图片后状态和预览同步更新。

## 风险与注意事项

1. 不要让附件区高度膨胀过快。
2. 总大小计算要与实际发送逻辑一致。
